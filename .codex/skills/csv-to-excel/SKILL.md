# CSV To Excel Converter

## Overview
This skill enables conversion of CSV files to Excel format with professional formatting, proper encoding handling (UTF-8, GBK), and multi-sheet support.

## Features
- **Automatic Encoding Detection**: Handles non-ASCII characters correctly.
- **Professional Formatting**: Bold headers, borders, auto-adjusted columns, and frozen panes.
- **Multi-Sheet Support**: Combine multiple CSVs into one Excel workbook.

## Usage
- "Convert this data.csv to Excel"
- "Combine these 3 report files into one Excel workbook"
