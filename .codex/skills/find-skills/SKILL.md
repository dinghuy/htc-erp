# Find Skills

This skill helps you discover and install specialized agent skills from the open ecosystem.

## When to Use
- When the user asks "how do I do X"
- When the user says "find a skill for X"
- When the user asks "can you do X"
- When the user wants to extend their agent's capabilities.

## Capabilities
- **Identify**: Find relevant skills based on user tasks or domains.
- **Search**: Search for tools, templates, or workflows.
- **Recommend**: Provide quality-based recommendations (GitHub stars, install count).
