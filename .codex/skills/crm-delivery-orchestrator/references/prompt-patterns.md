# Prompt Patterns

## Delivery Loop

- "Read Linear issue ABC-123 and the linked Notion spec, then outline the implementation task for crm-app."
- "Summarize the current release status from Linear and Notion for this CRM app."
- "Turn this Notion PRD into a bounded implementation task for crm-app."

## Design To Code

- "Use this Figma link to implement the quotation screen and verify the result with Playwright."
- "Compare this Figma node with the current frontend and tell me what is missing."

## QA And UAT

- "Run a browser verification of the lead flow and summarize UAT findings."
- "Check whether the current UI behavior matches the release checklist in Notion."

## Negative Examples

These should usually not trigger the skill:
- "Refactor this utility function."
- "Explain this TypeScript error in a local file."
- "Rename this component and update imports."
