# Buy Anything

Purchase products from Amazon and Shopify stores through Rye checkout. Like having a personal shopper in your chat app.

## Checkout Flow
1. **User provides product URL**.
2. **Collect shipping address**.
3. **Set up card via BasisTheory** (PCI-compliant secure vault).
4. **Submit order to Rye API**.

## Security
Card info never touches the chat — it goes directly to BasisTheory.
