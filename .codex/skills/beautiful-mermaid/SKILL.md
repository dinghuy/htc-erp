# Beautiful Mermaid

Render Mermaid diagrams as beautiful SVGs or ASCII art. Ultra-fast, fully themeable.

## When to Use
- User sends Mermaid code blocks (` ```mermaid ... ``` `)
- User asks to "render" or "visualize" a diagram
- User wants terminal/ASCII output for diagrams

## Supported Diagrams
Flowcharts, Sequence diagrams, Gantt charts, Entity Relationship diagrams, and more.

## Quick Start
- "Render this flowchart: [mermaid code]"
- "Show me this diagram in ASCII"
