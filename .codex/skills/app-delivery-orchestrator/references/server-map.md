# Server Map

- `Linear`: issue status, execution tracking, prioritization
- `Notion`: specs, PRDs, acceptance criteria, notes
- `Figma`: design source, screen/node context
- `Playwright`: browser reproduction and verification

## Precedence

- `Linear` wins for workflow state.
- `Notion` wins for requirements.
- `Figma` wins for visual intent when a file or node is provided.
- `Playwright` wins for observed browser behavior.
