# Prompt Patterns

- "Open the Linear issue, read the Notion spec, inspect the linked Figma node, then implement it."
- "Use the Figma node as source, align with Notion acceptance criteria, then build the UI."
- "Reproduce the bug in Playwright, compare with the intended behavior from Notion or Figma, then fix it."
- "Summarize release readiness using Linear issue state, Notion acceptance, and Playwright verification."
