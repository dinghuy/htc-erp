# Beaver Habit Tracker

Track and manage your daily habits using the Beaver Habit Tracker API.

## Tools
- **list_habits**: Show a weekly ASCII overview of your habits.
- **complete_habit**: Mark a habit as done or undone for a specific date.

## Setup
Requires `BEAVERHABITS_API_KEY` from settings.

## Usage
- "Show my habits for this week"
- "Mark 'Exercise' as done for today"
